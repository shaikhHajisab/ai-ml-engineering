# Module 01 — Python Fundamentals

## 🎯 Learning Objectives

By the end of this module, you will be able to:

1. Explain Python core data types and when to use each
2. Write Pythonic code using list comprehensions, generators, and decorators
3. Implement Object-Oriented Programming (OOP) in an ML context
4. Understand memory management and performance implications
5. Confidently answer Python questions in FAANG, startup, and ML engineer interviews

---

## 📋 Prerequisites

- Python installed (3.10+ recommended)
- VS Code or any code editor
- Git + GitHub account set up
